#### HTML

#### CSS
- 예시) 카카오게임즈가 운영하는 모든 ggb 구조는 어떤 페이지에 붙을지 알 수 없기 때문에 css간 충돌 발생을 막기 위하여 ggb.css 작성 방법을 언더스코어 대신 대쉬로 작성함
